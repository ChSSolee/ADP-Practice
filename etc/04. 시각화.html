산점도

  • main : 제목
  • xlab / ylab : 축 이름
  • xlim / ylim : 좌표축 범위
  • pch : 점의 종류 (0 ~ 18 : S언어 / 19 ~ 25 : 특수문자 / 32 ~ 127 : 아스키코드 )
  • cex : 점의 크기
  • col : 점의 색상
  • type : 그래프 종류 (p : 점 / l : 선 / b : 점과 선을 연결 / o : 점과 선을 중첩 / n : 그래프 초기화)
  • lty : 선 종류 (0 : 그리지 않음 / 1 : 실선 / 2 : 대시(-) / 3 : 점 / 4 : 점과 대시 / 5 : 긴 대시 / 6 : 두 개의 대시)
  • lwd : 선 굵기
library(MASS)
plot(Cars93$Length, Cars93$Weight, 
     main = "Cars93", # 제목 
     xlab = "Length", ylab = "Weight", # 축 이름
     xlim = c(130, 230), ylim = c(1695, 4105), # 좌표축 범위
     pch = 8, 
     # 점의 종류 
     cex = 1.5, # 점의 크기
     col = 50, # 점의 색상
     type = "b", # 그래프 종류 
     lty = 2, # 선 종류
     lwd = 2 # 선 굵기
     )

###### - 범례 - 위치 : “bottom”, “top”, “right”, “left”, “center”, 좌표 - pch : 점 그래프 범례 표시 형식 지정 - lty : 선 그래프 범례 표시 형식 지정 - fill : 면 그래프 범례 표시 형식 지정 - title : 범례 제목 지정 - bg : 범례의 배경 색 지정

plot(1:10, type="n", xlab=" ", ylab=" ")

legend("bottomright", c("x1","y1"), pch=1:2, title="bottomright")
legend("bottomleft", c("x2","y2"), pch=3:4, title="bottomleft")
legend("topleft", c("x3","y3"), pch=5:6, title="topleft")
legend("topright", c("x4","y4"), pch=7:8, title="topright")
legend("center", c("x5","y5"), lty=1:2, title="center")

legend(2.5,8,c("x6","y6"),lty=3:4,title="사용자 지정1")
legend(2.5,4,c("x7","y7"),lty=5:6,title="사용자 지정2")

- 직선 그래프 abline(a, b, h, v, coef, reg) - a : 기울기 - b : 절편 - h : y = h형태의 가로선 - v : x = v형태의 세로선 - coef : 기울기와 절편을 벡터 형태로 지정 - reg : 회귀식에 대한 그래프를 그릴 때 선형회귀모형을 지정

plot(cars, ylim=c(0,130), xlim=c(0,30), main="cars data")
cars_lm<-lm(dist~speed, data=cars)
abline(cars_lm, col="red")

abline(v = median(cars$speed), lty=3)
abline(h = median(cars$dist), lty=3)

- 곡선 그래프 curve(expr, from = …, to = … ) - expr : 그래프 표현할 식 - from : x축 시작점 - to : x축 끝점

curve(dnorm(x, mean=0, sd=1),
      from=-3, to=3,
      xlab="x", ylab="density", main="curve of dnorm")

- 막대 그래프 barplot(height, names.arg, space, horiz, main, xlab, ylab, col, beside, xlim, ylim) - beside : 범주가 여러 개일 경우의 표현 (TRUE : 여러 범주를 병렬로 / FALSE : 누적 막대)

- 히스토그램 hist(x, main, xlab, ylab, breaks, freq, col) - breaks : 막대의 구간 - freq : y축에 구간별 데이터의 개수(빈도)를 표시할지의 여부 (값을 지정하지 않거나 TRUE로 지정할 경우 : y축에 빈도가 표시 / FALSE : y축에 구간별 데이터의 확률밀도)
